## Revenue Categories

### Registration / Ticket Sales

| Line Item | Unit | Notes |
|---|---|---|
| Early-bird tickets | Per ticket | Typically 15-20% discount, 20-25% of registrations |
| Standard tickets | Per ticket | Full price |
| VIP / premium tickets | Per ticket | 2-3x standard price |
| Group discounts | Per group | Typically 10-15% for 5+ tickets |
| Academic / student tickets | Per ticket | 50-70% discount |
| Complimentary tickets | Per ticket | Speakers, sponsors, VIPs — budget 10-15% of total |

### Sponsorship Revenue

| Line Item | Unit | Notes |
|---|---|---|
| Platinum / Title sponsor | Per package | Typically 1-2 sponsors |
| Gold sponsor | Per package | Typically 3-5 sponsors |
| Silver sponsor | Per package | Typically 5-10 sponsors |
| Bronze / Community sponsor | Per package | Variable count |
| Session/track sponsorship | Per session | Naming rights for sessions |
| Networking sponsorship | Per activation | Lunch, reception, coffee break |

### Exhibitor Revenue

| Line Item | Unit | Notes |
|---|---|---|
| Booth rental (standard) | Per booth | By m² or standard size |
| Booth rental (premium/corner) | Per booth | 20-50% premium |
| Electrical / internet per booth | Per booth | Often separate charge |
| Lead scanning license | Per exhibitor | If provided by organizer |
| Show guide listing | Per exhibitor | Print or digital directory |

### Other Revenue

| Line Item | Unit | Notes |
|---|---|---|
| Workshop fees | Per workshop | If priced separately |
| Merchandise sales | Per item | T-shirts, books, etc. |
| On-demand content access | Per license | Post-event recordings |

## Fixed Cost Categories

These costs do not change with attendee count. Expressed as % of the **Venue & F&B** or **Technology** category budget from `cost-benchmarks.md`.

| Line Item | Unit | % of Its Category | Notes |
|---|---|---|---|
| Venue rental | Per day | 35-50% of Venue & F&B | The single largest line item |
| AV base package (included in venue) | Flat fee | 0-15% of Technology | Often included in venue deal |
| Event platform license (virtual/hybrid) | Per event | 20-40% of Technology | Virtual: higher share |
| Event insurance | Per event | 1-3% of total budget | Non-negotiable |
| Permits and licenses | Per event | 0-1% of total budget | Varies by jurisdiction |
| Event website (design + hosting) | Per event | 15-30% of Marketing | Foundation — required |
| Event app license | Per event | 10-25% of Technology | Depends on features |
| Project management tools | Per event | <1% of total budget | Often existing tools |

## Variable Cost Categories

These costs scale directly with attendee count. Expressed as **relative ratios** (all-day catering package = 1.0×).

| Line Item | Unit | Ratio (all-day catering = 1.0×) | Notes |
|---|---|---|---|
| Catering — coffee breaks | Per person/per break | 0.1-0.15× | Per break |
| Catering — lunch | Per person/per day | 0.4-0.6× | Buffet; plated = 0.6-0.9× |
| Catering — dinner | Per person | 0.9-1.3× | Wine pairing adds 30-50% |
| Catering — reception | Per person | 0.5-0.7× | Drinks + canapés |
| Name badges / lanyards | Per person | 0.03-0.1× | |
| Printed program / agenda | Per person | 0.04-0.12× | Consider digital alternative |
| Swag bag / gifts | Per person | 0.1-0.6× | Wide range by quality |
| WiFi bandwidth per device | Per person | 0.03-0.12× | |
| Welcome pack | Per person | 0.06-0.25× | |
| Certificate of attendance | Per person | 0.01-0.06× | |

**To get absolute prices:** Get one local catering quote for an all-day conference package (2 coffee breaks + lunch). That's your 1.0× anchor. Multiply by the ratios above.

## Contingency Rules

- **Standard events:** 10% of total expenses
- **First-time events:** 15% of total expenses
- **Hybrid events:** 15-20% (more failure points)
- **International events:** 15% + 3-5% currency buffer
- **Outdoor events:** 15% + weather contingency

Never present a budget without a contingency line. Contingency is not a slush fund — it covers genuine unknowns and overruns.

## Cost Cutting Priority Order

When budget is tight, cut in this order (least impact to most impact):

1. **Swag / gifts** — Nice to have, not essential
2. **Printed materials** — Use digital alternatives
3. **Professional photographer** — Use staff phones for basic documentation
4. **Entertainment / DJ** — Background music via playlist
5. **Premium catering** — Downgrade from plated to buffet, cut dinner
6. **Paid speakers** — Use practitioners willing to speak for exposure
7. **Promotional video** — Use static graphics and photos
8. **Stage design** — Simplify to podium + screen
9. **Venue** — Downgrade to smaller or less premium space

**Never cut:**
- Contingency
- Basic AV (sound, projection)
- WiFi (for tech audiences)
- Speaker travel support
- Post-event follow-up budget
