## Platform Rules

Apply these constraints when generating posts:

| Element | LinkedIn | Twitter/X | Instagram |
|---|---|---|---|
| Max length | ~3,000 chars; aim for 150-300 | 280 chars | 2,200 chars; aim for <125 for feed |
| Hook placement | First 2 lines (before "see more") | First sentence | First line of caption |
| Links | In first comment, NOT body (algorithm penalty) | In tweet body | Link in bio only; use "link in bio" CTA |
| Hashtags | 1-3 per post (relevant, not spammy) | 2-3 per tweet | 5-10 (mix branded + industry) |
| Best format | PDF carousels, native video (15-60s), text-only thought leadership | Short text, threads, quote tweets | Reels (20-45s), Stories, carousels |
| Posting frequency | 2-3x/week promotion; daily during event | 3-5 daily during event | 3-5x/week feed; daily Stories during event |
| Tagging | Tag speakers and sponsors by name/handle | @ mention speakers | @ mention in post and photo tags |
| Best time (B2B) | Tue-Thu, 8-10 AM | Tue-Wed, 9-11 AM | Mon-Fri, 10 AM-1 PM |

## Pre-Event Templates

### Speaker Spotlight (LinkedIn)

```
[Hook line: insight or question related to speaker's topic]

We're thrilled to announce that {{speaker_name}}, {{speaker_title}} at {{speaker_company}}, will be taking the stage at {{event_name}}.

Session: "{{session_title}}"

[2-3 sentences: what attendees will learn, why it matters now]

📅 {{event_date_display}}
📍 {{venue_city}}

Seats are filling fast.

👉 Register: [link in first comment]

#{{event_hashtag}} #[industry_hashtag]
```

### Speaker Spotlight (Twitter/X)

```
🎤 Speaker announcement!

{{speaker_name}} ({{speaker_title}}, {{speaker_company}}) joins {{event_name}} to talk {{session_topic_short}}.

{{event_date_display}} | {{venue_city}}

Register → {{registration_link}}

#{{event_hashtag}}
```

### Countdown Post (LinkedIn)

```
{{days_until_event}} days until {{event_name}} 🔥

Here's what we're most excited about:

[3-4 bullet points: specific sessions, speakers, experiences]

{{registered_count}}+ professionals are already registered. Will you be there?

👉 Link in comments

#{{event_hashtag}}
```

### Session Teaser (LinkedIn — carousel format)

```
Struggling with [problem the session addresses]?

At {{event_name}}, {{speaker_name}} will show you [specific solution or framework].

You'll walk away with:
→ [Takeaway 1]
→ [Takeaway 2]
→ [Takeaway 3]

[Swipe through for the full session breakdown ➡️]

📅 {{event_date_display}} | 📍 {{venue_city}}

#{{event_hashtag}}
```

### Registration Urgency (Twitter/X)

```
Only {{seats_remaining}} spots left for {{event_name}}.

{{registered_count}} {{audience_descriptor}} registered.
{{number_of_sessions}} sessions. {{number_of_speakers}} speakers.

Don't wait → {{registration_link}}

#{{event_hashtag}}
```

### Sponsor Highlight (LinkedIn)

```
We're proud to welcome {{sponsor_name}} as a {{sponsor_tier}} sponsor of {{event_name}}.

[1-2 sentences: what the sponsor does and why it's relevant to the event audience]

Visit them at [booth number / virtual booth] to [learn about / demo / discuss] [specific value].

Thank you, {{sponsor_name}}, for making {{event_name}} possible.

#{{event_hashtag}} #[sponsor_hashtag]
```

## During-Event Templates

### Live Speaker Quote (LinkedIn + Twitter/X)

**LinkedIn:**
```
💬 "{{speaker_quote}}"

— {{speaker_name}}, {{speaker_title}} at {{speaker_company}}

Live now at {{event_name}}.

#{{event_hashtag}}
```

**Twitter/X:**
```
"{{speaker_quote_short}}"

— {{speaker_name}} at #{{event_hashtag}}
```

Tip: Break keynotes into 2-3 quote posts. Use branded graphics with speaker photo for LinkedIn. Text-only for Twitter/X for speed.

### Session Recap Thread (Twitter/X)

```
Tweet 1: 🧵 Key takeaways from "{{session_title}}" by {{speaker_name}} at #{{event_hashtag}}

Tweet 2: 1/ [First key insight — one clear sentence]

Tweet 3: 2/ [Second key insight]

Tweet 4: 3/ [Third key insight]

Tweet 5: Full recording coming soon. Follow us for updates. #{{event_hashtag}}
```

### Attendee Highlight (Instagram Story / LinkedIn)

```
"[Attendee quote about their experience]"

— {{attendee_name}}, {{attendee_company}}

[Photo of attendee at event]

#{{event_hashtag}}
```

## Content Mix

Recommended content mix for pre-event promotion:

| Category | % of Posts | Purpose |
|---|---|---|
| Speaker/session spotlights | 30% | Build content credibility |
| Registration CTAs | 20% | Drive conversions |
| Behind-the-scenes | 15% | Build anticipation, humanize the event |
| Sponsor highlights | 15% | Sponsor value delivery |
| Countdown/urgency | 10% | Time pressure |
| UGC/engagement | 10% | Community involvement |

Post at least 2x per week for 8-12 weeks pre-event. Increase to daily in the final 2 weeks.

## Alternative LinkedIn Structures

Not every post should follow the [Hook → Details → Bullets → CTA] pattern. Using the same structure repeatedly is the fastest way to get flagged as AI-generated content. Vary with these alternatives and use each at least once per promotion cycle.

### "Hot Take" Format
```
[Contrarian or surprising opening — challenges conventional wisdom]

[2-3 sentences of reasoning — why the contrarian view has merit]

[Bridge to event — specific session or speaker that addresses this tension]

[Soft CTA in comments, not in body]
```

### "Story" Format
```
[Personal anecdote or observation — something that happened, not something you read]

[What it taught you — one clear insight, not a list]

[Connection to event — this is exactly what Session X covers]

[CTA]
```

### "Data First" Format
```
[Striking statistic or number — standalone line, no context yet]

[What it means — 1-2 sentences of interpretation]

[How the event addresses this — specific session, speaker, or workshop]

[CTA]
```

### "Question" Format
```
[Open question the audience actually wrestles with — not rhetorical fluff]

[Brief acknowledgment that it is genuinely hard / complex / debated]

[Tease that the event tackles this — name the session or panel]

[CTA]
```
