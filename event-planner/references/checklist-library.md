## Phase 1: Concept

### Strategy & Objectives
- [ ] Define event purpose statement (one sentence: why this event exists)
- [ ] Set 2-4 measurable objectives with specific KPIs
- [ ] Identify target audience and estimate realistic attendance
- [ ] Choose event type and format (in-person / hybrid / virtual, single / multi-day)
- [ ] Determine number of tracks or parallel sessions (if applicable)
- [ ] Get executive sponsor sign-off on objectives and budget envelope
- [ ] Create event brief / planning canvas document

### Budget
- [ ] Define total budget envelope
- [ ] Allocate budget across major categories (venue, F&B, AV, marketing, speakers, contingency)
- [ ] Set ticket pricing strategy (if applicable)
- [ ] Identify revenue sources (tickets, sponsors, exhibitors, internal funding)
- [ ] Reserve 10-15% contingency buffer
- [ ] Set up budget tracking spreadsheet or tool

### Date & Venue
- [ ] Research potential dates (check holidays, competitor events, internal calendar)
- [ ] Shortlist 3-5 venue options
- [ ] Request venue proposals with pricing, capacity, and available dates
- [ ] Conduct site visits for top 2-3 venues
- [ ] Evaluate venues against criteria (capacity, accessibility, AV, catering, location, cost)
- [ ] Negotiate venue contract (cancellation terms, minimum spend, comp rooms)
- [ ] Sign venue contract and confirm date(s)

### Team
- [ ] Identify event lead (single point of accountability)
- [ ] Assemble core team and assign workstream responsibilities
- [ ] Create RACI matrix for major workstreams
- [ ] Set up project management tool and communication channel
- [ ] Schedule recurring planning meetings (weekly recommended)

## Phase 2: Planning

### Content & Speakers
- [ ] Define content strategy and themes aligned with objectives
- [ ] Decide speaker sourcing approach (CFP, invitation, or hybrid)
- [ ] If CFP: draft CFP copy, set submission deadline, choose review platform
- [ ] If invitation: create speaker target list with backup options
- [ ] Open CFP or begin speaker outreach
- [ ] Review submissions / receive speaker confirmations
- [ ] Select sessions and build preliminary agenda
- [ ] Send speaker confirmation letters with expectations and deadlines
- [ ] Collect speaker bios, headshots, and session abstracts
- [ ] Plan session formats (keynote duration, panel setup, workshop logistics)
- [ ] Identify moderators for panels and Q&A sessions

### Registration
- [ ] Select registration platform (or configure existing tool)
- [ ] Design registration form (essential fields only — minimize friction)
- [ ] Set up ticket types and pricing tiers (early-bird, standard, VIP, group)
- [ ] Configure confirmation emails and receipts
- [ ] Test registration flow end-to-end before launch
- [ ] Ensure GDPR/data protection compliance (privacy notice, consent checkboxes, data processing agreement)
- [ ] Set up attendee data export and CRM integration
- [ ] Launch registration

### Marketing & Promotion
- [ ] Develop event brand identity (name, logo, color palette, visual language)
- [ ] Build event website with key information (date, location, agenda, speakers, registration)
- [ ] Create content calendar for pre-event promotion
- [ ] Draft and schedule email campaign sequence (save-the-date → early-bird → speaker announcements → last chance)
- [ ] Prepare social media content and graphics
- [ ] Set up event hashtag and social media monitoring
- [ ] Plan paid promotion if budget allows (LinkedIn ads for B2B, Google Ads)
- [ ] Brief PR/communications team (if media attendance expected)
- [ ] Create referral or ambassador program (if applicable)

### Sponsors & Exhibitors
- [ ] Define sponsorship tiers and benefits packages
- [ ] Create sponsorship prospectus document
- [ ] Build target sponsor list with contact owners
- [ ] Begin sponsor outreach (start with highest tiers)
- [ ] Negotiate and close sponsor agreements
- [ ] Collect sponsor deliverables (logos, descriptions, collateral)
- [ ] If trade show: design floor plan and booth allocation process
- [ ] Create exhibitor guide with technical specifications

### Logistics & Vendors
- [ ] Book AV/production vendor (stage, screens, sound, lighting)
- [ ] Book catering (or confirm venue in-house catering)
- [ ] Book photographer and/or videographer
- [ ] Arrange accommodation block for speakers and team (negotiate group rate)
- [ ] Plan attendee transportation (if venue requires shuttle or is hard to reach)
- [ ] Book entertainment or team-building activities (if applicable)
- [ ] Research and book WiFi upgrade (if venue default is insufficient)
- [ ] Plan power and connectivity for exhibitor booths (if applicable)
- [ ] Source badge printing solution (on-site printing or pre-printed)

## Phase 3: Execution Prep

### Agenda & Run-of-Show
- [ ] Finalize complete agenda with exact session times, rooms, and speakers
- [ ] Build run-of-show document (minute-by-minute schedule for production team)
- [ ] Assign session hosts or room managers for each room
- [ ] Plan transition times between sessions (minimum 15 minutes for room changes)
- [ ] Schedule speaker rehearsals (especially for keynotes and demos)
- [ ] Prepare speaker ready room (quiet space with power, WiFi, water)

### Vendor Coordination
- [ ] Send final briefs to all vendors with confirmed specifications
- [ ] Confirm catering final numbers (usually required 72h before event)
- [ ] Confirm AV setup specifications per room
- [ ] Share load-in/load-out schedule with venue and vendors
- [ ] Distribute emergency contact list to all vendors

### Materials & Production
- [ ] Finalize attendee badge design and data
- [ ] Print or configure on-site badge printing
- [ ] Produce signage (wayfinding, room names, sponsor acknowledgment, Wi-Fi info)
- [ ] Prepare printed programs or digital agenda app
- [ ] Assemble attendee welcome packs (if applicable)
- [ ] Order event swag or gifts (if applicable)
- [ ] Prepare speaker gifts (if applicable)

### Communications
- [ ] Send "what to expect" email to attendees (logistics, directions, parking, dress code)
- [ ] Send final speaker briefing email (schedule, AV details, arrival time, contact person)
- [ ] Brief all event staff and volunteers (roles, schedule, escalation process)
- [ ] Prepare on-site communication plan (radios, WhatsApp group, Slack channel)
- [ ] Draft social media posts for event days (pre-write and schedule where possible)

### Technology
- [ ] Test registration/check-in system
- [ ] Test WiFi under expected load (or confirm venue guarantee)
- [ ] Test streaming setup (if hybrid/virtual component)
- [ ] Test event app (if using one)
- [ ] Prepare backup plans for tech failures (offline check-in list, backup presentations on USB)

### Risk & Contingency
- [ ] Review and update risk register
- [ ] Confirm insurance coverage
- [ ] Prepare emergency procedures document (evacuation routes, first aid, emergency contacts)
- [ ] Identify backup speakers for keynote slots
- [ ] Confirm cancellation/postponement procedures with venue and vendors

## Phase 4: Event Day

### Setup (Day Before or Morning Of)
- [ ] Venue walkthrough — verify room setups match run-of-show
- [ ] Install signage and wayfinding
- [ ] Test all AV in every session room
- [ ] Set up registration/check-in desk
- [ ] Set up speaker ready room
- [ ] Set up sponsor/exhibitor area (or oversee exhibitor self-setup)
- [ ] Verify catering setup and timing
- [ ] Brief venue staff on event flow and key contacts
- [ ] Distribute radios or confirm communication channel for event team

### During Event
- [ ] Open registration desk (90 min before first session minimum)
- [ ] Welcome attendees, distribute badges and materials
- [ ] Monitor session timekeeping (signal speakers at 5-min and 1-min warnings)
- [ ] Manage room changes and transitions
- [ ] Post real-time social media updates
- [ ] Capture photos throughout the day
- [ ] Monitor and resolve issues as they arise (escalation path clear)
- [ ] Track attendance per session (for analytics and fire code compliance)
- [ ] Coordinate catering service timing (breaks, lunch, reception)
- [ ] Support sponsors and exhibitors
- [ ] Facilitate networking moments (if structured networking planned)

### End of Day
- [ ] Secure equipment and materials overnight (if multi-day)
- [ ] Back up all recordings and photos
- [ ] Quick team debrief (15 min max — what worked, what to fix for tomorrow)
- [ ] Confirm next-day adjustments (room changes, timing shifts, catering tweaks)

## Phase 5: Post-Event

### Immediate (T+1 to T+3 days)
- [ ] Send thank-you email to all attendees (within 24h)
- [ ] Send personalized thank-you to speakers (within 24h)
- [ ] Send personalized thank-you to sponsors (within 24h)
- [ ] Distribute post-event survey (within 48h — response rate drops sharply after 72h)
- [ ] Share event photos on social media and with attendees
- [ ] Export final attendee data and session attendance records
- [ ] Hand off qualified leads to sales team with engagement context

### Short-term (T+1 to T+2 weeks)
- [ ] Process and edit session recordings
- [ ] Publish session recordings and slide decks (if sharing publicly)
- [ ] Write event recap blog post or newsletter
- [ ] Analyze survey results and create summary report
- [ ] Begin financial reconciliation (collect all invoices, compare actuals to budget)
- [ ] Send final sponsor reports with deliverable proof and engagement data

### Medium-term (T+2 to T+4 weeks)
- [ ] Complete financial reconciliation and close event budget
- [ ] Conduct internal debrief meeting with full planning team
- [ ] Compile ROI report against original objectives and KPIs
- [ ] Document lessons learned (what worked, what to change, what to keep)
- [ ] Archive all event assets (designs, templates, vendor contacts, contracts)
- [ ] Begin planning for next edition (if recurring event)
- [ ] Send final sponsor invoices and process outstanding payments

## Scale Adjustments

### Small events (<100 attendees)
Remove or simplify: exhibitor management, multi-track scheduling, sponsor tiers, badge printing (use name tags), formal run-of-show. Keep: objectives, timeline, attendee comms, post-event survey.

### Medium events (100-500 attendees)
Use full checklist but adjust scope: 2-3 sponsor tiers instead of 4, 1-2 tracks instead of 3+, simplified registration flow.

### Large events (500-2,000 attendees)
Use complete checklist. Add: dedicated registration team, volunteer management, security coordination, WiFi load testing, overflow room planning.

### Very large events (2,000+ attendees)
Use complete checklist plus: multi-venue coordination, transportation management, crowd flow simulation, dedicated medical support, government permits (if applicable), professional security team.
