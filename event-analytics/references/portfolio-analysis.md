## Event Portfolio Scoring Framework

Score each event (past or planned) on these five dimensions. Use to prioritize which events to continue, scale, modify, or cut.

| Dimension | Weight | Score 1-5 | How to Assess |
|---|---|---|---|
| Strategic alignment | 25% | | Does this event serve a stated business objective? (Brand, pipeline, retention, recruitment) |
| Audience quality | 25% | | Are the right people there? (Decision-makers, ICP match, seniority level) |
| ROI / cost efficiency | 20% | | Pipeline per unit invested? Leads per unit spent? Revenue attributed? |
| Reach and visibility | 15% | | How many people does this event expose us to? (Attendees + social + media) |
| Competitive presence | 15% | | Are competitors there? Would absence be noticed? |

**Scoring guide:**
- 5 = Top quartile. Clear, measurable impact. Would be painful to cut.
- 4 = Strong performer. Good returns, but not irreplaceable.
- 3 = Average. Delivers some value, but could be replaced.
- 2 = Below average. Questionable ROI. Needs restructuring or budget cut.
- 1 = Poor. No measurable impact. Cut unless strategic reasons override.

**Weighted score = Σ (Score × Weight). Max = 5.0.**
- 4.0+ = Invest more. Scale up.
- 3.0-3.9 = Maintain. Optimize for efficiency.
- 2.0-2.9 = Review. Reduce scope or budget.
- Under 2.0 = Cut. Redirect budget to higher-scoring events.

**Normalization rules:**
- Compare CPL (cost per lead) across events — it's the most comparable metric
- Pipeline ROI requires consistent attribution windows (use 180 days for all)
- For brand events (no lead gen goal): use reach metrics instead (impressions, social mentions, media coverage)
- For webinars: count live + on-demand combined as "attendees"
- For sponsored/exhibited events: only count YOUR leads, not total event attendance

**Common mistake:** Comparing a 200K conference with a 2K webinar on absolute pipeline. Use per-unit-spent metrics. The webinar might generate less pipeline total but at 10x the efficiency.

- Avoid Q4 overload (Oct-Nov is prime conference season — don't stack)
- Spread owned events across quarters for consistent pipeline generation
- Buffer: no two owned events within 4 weeks of each other (team burnout)
- Webinar series: consistent cadence (monthly or bi-weekly), pause in Aug and late Dec
- Trade shows are fixed dates — plan owned events around them, not vice versa

## Budget Allocation Across Events

### Typical B2B Event Budget Split (by event type)

| Event Type | % of Total Event Budget | Rationale |
|---|---|---|
| Annual owned conference | 25-40% | Flagship. Highest ROI if executed well. |
| Trade show / expo participation | 20-30% | High visibility. Often contractually committed. |
| Webinar program | 5-10% | Low cost, high volume. Steady pipeline. |
| Executive / VIP events | 5-10% | High value per attendee. Relationship ROI. |
| Roadshow / field events | 10-15% | Regional coverage. Sales team alignment. |
| Sponsoring other events | 10-15% | Access to audiences you can't build alone. |
| Internal events (SKO, Town Hall) | 5-10% | Often separate budget from marketing events. |

**Decision framework for adding/cutting events:**
- Adding: "Will this event reach an audience we can't reach through existing events?"
- Cutting: "If we skip this, will anyone notice? Will it impact pipeline in the next 6 months?"
- Scaling: "Is this event at capacity? Could we add a second edition (different city, different quarter)?"
