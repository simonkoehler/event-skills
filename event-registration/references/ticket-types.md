# Ticket Types

## Standard Tier Models

### 3-Tier Model (most common)

| Element | Standard | Premium / VIP | Executive |
|---|---|---|---|
| Price | Base | 1.5-2x base | 2.5-3x base |
| Sessions | All main sessions | All + workshops | All + executive roundtables |
| Networking | General | VIP reception | 1:1 scheduler + dinner |
| Content | Recordings (on-demand) | Recordings + slides | Recordings + slides + exclusive |
| Support | General | Priority | Concierge |
| Badge | "Most Popular" label | — | — |
| Availability | Open | Limited | Very limited (10-20) |

### 2-Tier Model (simpler events)

| Element | General | VIP |
|---|---|---|
| Price | Base | 2x base |
| Access | All sessions | All + exclusive experiences |

### Day Pass Model (multi-day events)

| Element | Single Day | Full Event | Full + Workshop |
|---|---|---|---|
| Price | 40-50% of full | Base | 1.3x base |

## Pricing Timeline

| Phase | Timing | Price | Notes |
|---|---|---|---|
| Super early-bird | T-8 to T-6 months | 25-30% off | First 50-100 tickets only |
| Early-bird | T-6 to T-3 months | 15-20% off | Until date or allocation exhausted |
| Standard | T-3 months to T-1 week | Full price | Main selling window |
| Late / on-site | T-1 week to event day | 10-20% premium | Urgency premium |

## Discount Types

| Type | Discount | Conditions |
|---|---|---|
| Early-bird | 15-20% | Time-limited or quantity-limited |
| Group (5+) | 10-15% | Same company, single purchase |
| Group (10+) | 15-20% | Same company, single purchase |
| Academic / student | 50-70% | Valid ID required |
| Non-profit | 20-30% | Organization verification |
| Alumni (returning attendee) | 10-15% | Attended previous edition |
| Speaker referral | 10-15% | Code from confirmed speaker |
| Promotional | Varies | Limited-time campaign codes |

## Complimentary Tickets

Budget 10-15% of capacity as complimentary for standard B2B conferences.

**Customer / User Conference exception:** For events with customer retention as a primary objective (SaaS user conferences, partner summits where customers are key audience), comp pool runs much higher — typically 30-50% for existing customers (one-time annual benefit, builds retention + advocacy) + 5-10% for prospects. The remaining ~40-65% goes to paid tickets. Budget revenue projection accordingly: a 500-pax customer conference with 40% customer comp may only generate paid revenue from ~250-300 tickets.

Standard comp allocation for B2B conferences:

| Recipient | Tickets | Notes |
|---|---|---|
| Speakers | 1 per speaker | Full event pass |
| Sponsors | Per sponsorship package | Platinum: 10, Gold: 5, Silver: 3 |
| Press / media | 5-15 | Pre-approved media list |
| Advisory board | 3-5 | If applicable |
| VIP guests | 5-10 | Executive invitations |
| Staff / volunteers | As needed | Working passes |
