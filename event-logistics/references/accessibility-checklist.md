# Accessibility Checklist

Load this reference when evaluating venues, planning on-site operations, or designing registration forms. Accessibility is not optional — the European Accessibility Act (2025) establishes legal requirements, and attendees expect inclusive events regardless of jurisdiction.

## Physical Accessibility

- [ ] All event spaces wheelchair accessible (no steps-only access)
- [ ] Accessible restrooms on same floor as event rooms
- [ ] Ramps or lifts for stage access (speakers with mobility needs)
- [ ] Reserved seating: front row with aisle access in every session room
- [ ] Wide aisles between seating rows (minimum 90cm / 36 inches)
- [ ] Accessible registration desk (lowered counter section)
- [ ] Elevator access to all floors used for the event
- [ ] Accessible parking spaces close to venue entrance

## Sensory Accessibility

- [ ] Live captioning / CART for all plenary sessions
- [ ] Hearing loop (induction loop) in main session room
- [ ] Sign language interpreter available (book T-3 months if needed)
- [ ] High-contrast signage with large fonts (minimum 24pt for wayfinding)
- [ ] Good lighting levels in all session rooms (important for lip-reading)
- [ ] Microphones required for ALL speakers and Q&A (no "I'll just project my voice")
- [ ] Materials available in accessible digital formats (not just PDFs — screen reader compatible)

## Neurodivergent Needs

- [ ] Quiet room available throughout the event (low-light, low-noise)
- [ ] Sensory break schedule communicated (not just session breaks)
- [ ] Clear, predictable wayfinding (consistent signage, logical room numbering)
- [ ] Content warnings for sessions with flashing lights, loud sounds, or distressing topics
- [ ] Option to attend sessions virtually even at in-person events (reduces sensory overwhelm)

## Digital Accessibility

- [ ] Event website meets WCAG 2.1 AA standards
- [ ] Event app compatible with screen readers
- [ ] All images on website and materials have alt text
- [ ] Video content has captions
- [ ] Registration form keyboard-navigable

## Communication

- [ ] Registration form includes: "Please share any accessibility needs so we can support your participation"
- [ ] Dedicated accessibility contact person (name and email, not a generic inbox)
- [ ] Accessibility information published on event website BEFORE registration opens
- [ ] Dietary needs form covers allergies with severity levels (not just "vegetarian/vegan")
- [ ] Speaker briefing includes accessibility reminders (describe visual content, use mic, face audience)

## Venue Evaluation Addition

When scoring venues (see venue-checklist.md), weight accessibility at minimum 10% — not 5%. A venue that fails basic accessibility excludes attendees, creates legal risk, and damages reputation. Non-negotiable requirements: wheelchair access to all spaces, accessible restrooms, hearing loop capability.
