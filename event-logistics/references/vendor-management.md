## Vendor RFP Template

```
# Request for Proposal: {{vendor_category}} Services
## {{event_name}} — {{event_date_display}}, {{venue_city}}

### Event Overview
{{event_name}} is a {{event_type}} for {{expected_attendees}} attendees
taking place on {{event_date_display}} at {{venue_name}}, {{venue_city}}.
[2-3 sentences: event description, audience, format]

### Scope of Work
We are seeking a {{vendor_category}} vendor to provide:
- [Specific deliverable 1 with quantity/spec]
- [Specific deliverable 2]
- [Specific deliverable 3]

### Detailed Requirements
[Room-by-room or item-by-item specifications]

### Timeline
| Milestone | Date |
|---|---|
| RFP response deadline | {{rfp_deadline}} |
| Vendor selection | {{selection_date}} |
| Contract signing | {{contract_date}} |
| Site visit / tech survey | {{site_visit_date}} |
| Load-in | {{load_in_date}} |
| Event dates | {{event_date_display}} |
| Load-out / teardown | {{teardown_date}} |

### What to Include in Your Response
1. Company overview and relevant experience
2. Proposed solution (equipment list, staffing plan, approach)
3. Itemized pricing (including tax, delivery, setup, teardown)
4. References from 2-3 similar events in the past 12 months
5. Insurance certificate (minimum public liability coverage)
6. Cancellation and change policy

### Evaluation Criteria
| Criterion | Weight |
|---|---|
| Technical capability and proposed solution | 30% |
| Price and value | 25% |
| Relevant experience and references | 20% |
| Flexibility and responsiveness | 15% |
| Insurance and compliance | 10% |

### Contact
Questions: {{contact_name}} — {{contact_email}}
Response deadline: {{rfp_deadline}}
Submit to: {{submission_email}}
```

## Vendor Brief Template

Sent to confirmed vendors 2-4 weeks before event.

```
# Vendor Brief: {{vendor_category}}
## {{event_name}} — {{event_date_display}}

### Event Summary
- Event: {{event_name}}
- Dates: {{event_date_display}}
- Venue: {{venue_name}}, {{venue_address}}
- Expected attendees: {{expected_attendees}}
- Format: {{event_format}}

### Your Scope
[Confirmed deliverables — reference contract/PO number]

### Key Dates
| Date | Activity |
|---|---|
| {{load_in_date}} {{load_in_time}} | Load-in begins |
| {{setup_complete_time}} | Setup must be complete |
| {{event_day_1}} | Event day 1 |
| {{event_day_2}} | Event day 2 (if multi-day) |
| {{teardown_start}} | Teardown begins |
| {{teardown_complete}} | Venue must be cleared |

### Venue Access
- Loading dock: {{loading_dock_location}}
- Freight elevator: {{freight_elevator_details}}
- Vendor parking: {{vendor_parking}}
- Venue contact: {{venue_contact_name}}, {{venue_contact_phone}}

### On-Site Contact
- Event Manager: {{event_manager_name}}, {{event_manager_phone}}
- Logistics Lead: {{logistics_lead_name}}, {{logistics_lead_phone}}
- Communication channel: {{comms_channel}} (radio channel / WhatsApp group)

### Run-of-Show Extract
[Relevant sections from the run-of-show that affect this vendor]

### Special Notes
[Any venue-specific rules, access restrictions, union requirements,
power limitations, or timing constraints]

### Emergency Procedures
- Emergency contact: {{emergency_contact}}
- Nearest hospital: {{nearest_hospital}}
- Evacuation assembly point: {{assembly_point}}
```

## Common Vendor Categories

### Essential (every in-person event needs these)

| Category | What they provide | Book by |
|---|---|---|
| Venue | Space, basic infrastructure | T-9 to T-12 months |
| AV / Production | Sound, projection, lighting, staging | T-6 months |
| Catering | Food and beverages | T-6 months (or venue in-house) |

### Standard (most conferences need these)

| Category | What they provide | Book by |
|---|---|---|
| Registration / check-in | Badge printing, scanning, desk staff | T-3 months |
| Photography | Event documentation, speaker shots | T-3 months |
| Signage / print | Wayfinding, banners, programs | T-4 weeks (production) |
| WiFi / internet | Dedicated bandwidth, network management | T-3 months |

### Situational (depends on event type and scale)

| Category | When needed | Book by |
|---|---|---|
| Videography / streaming | Hybrid events, content capture | T-4 months |
| Security | Large events (500+), VIP events | T-2 months |
| Transportation | Remote venues, VIP shuttles | T-2 months |
| Simultaneous translation | International events | T-3 months |
| Florist / décor | Galas, premium events | T-2 months |
| Entertainment | Dinners, receptions, SKOs | T-3 months |
| Temporary staffing | Large events, registration rush | T-1 month |
