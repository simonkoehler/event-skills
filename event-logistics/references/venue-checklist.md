## Venue Scorecard

Rate each criterion 1-5 and multiply by weight for weighted score.

| Category (Weight) | Criteria | Questions to Assess |
|---|---|---|
| **Capacity & Layout (20%)** | Total capacity | Does it fit expected attendance + 20% buffer? |
| | Room count | Enough rooms for all tracks + speaker prep + registration? |
| | Flexible config | Can rooms be set up theater, classroom, banquet, U-shape? |
| | Breakout spaces | Separate rooms for parallel sessions, workshops? |
| | Stage area | Adequate for keynote production (lighting, screens)? |
| | Foyer / common area | Space for networking, sponsor booths, coffee breaks? |
| **Location & Access (20%)** | Transport links | Proximity to airport, train station, public transit? |
| | Parking | On-site parking? How many spaces? Cost? |
| | Hotel proximity | Walking distance hotels? Room block negotiable? |
| | Walkability | Restaurants, shops for attendees during breaks? |
| | International access | Easy visa situation? Direct flights from key cities? |
| **Technology & AV (15%)** | Built-in AV | Projectors, screens, sound systems included? Quality? |
| | WiFi capacity | Concurrent device support? Dedicated bandwidth available? |
| | Power outlets | Sufficient per room? Floor boxes? Extension allowance? |
| | Streaming capability | Dedicated upload line for hybrid? Encoding room? |
| | Mobile signal | Coverage throughout venue including basement/underground? |
| **Catering (15%)** | In-house vs. external | Must use in-house caterer or can bring preferred vendor? |
| | Quality and variety | Dietary options (vegan, halal, gluten-free, allergies)? |
| | Service style | Buffet, plated, food stations, cocktail all possible? |
| | Minimum spend | F&B minimum commitment? |
| | Alcohol policy | Licensed? Corkage fee? Responsible service policy? |
| **Cost & Value (15%)** | Rental rate | Day rate or half-day rate? Weekend vs. weekday? |
| | What's included | AV, WiFi, tables/chairs, linens, cleaning, security? |
| | Hidden fees | Service charge, corkage, overtime, cleaning, damage deposit? |
| | Negotiation room | Multi-day discount? Off-peak discount? Package deals? |
| **Logistics (10%)** | Loading dock | Size, access hours, ramp or lift? |
| | Freight elevator | Capacity, dimensions, availability during setup? |
| | Setup time | How much time before event for load-in? After for teardown? |
| | Storage | Secure storage for equipment overnight (multi-day events)? |
| | Vendor access | Separate entrance for vendors? Early access? |
| **Accessibility (5%)** | Wheelchair access | All areas accessible? Ramps, lifts, accessible restrooms? |
| | Hearing assistance | Hearing loop, captioning capability? |
| | Visual | Clear signage, good lighting, high contrast? |
| | Gender-neutral facilities | Available? |
| | Quiet room | Available for attendees who need a break? |

## Site Visit Checklist

Walk through these during every venue visit:

### Before Arriving
- [ ] Bring event brief with room requirements, capacity, and AV needs
- [ ] Prepare list of specific questions from scorecard gaps
- [ ] Take a measuring tape for room dimensions
- [ ] Plan to visit at the same time of day as event (check natural light, noise)

### During the Visit
- [ ] Walk the attendee journey: entrance → registration → main room → breakout → catering → restrooms
- [ ] Test WiFi speed on your phone (download and upload)
- [ ] Check mobile signal in all rooms
- [ ] Note column placements that could block sight lines
- [ ] Measure ceiling height (affects screen and projection size)
- [ ] Check noise bleed between rooms (close doors, listen)
- [ ] Assess natural light (good for energy, bad for projectors)
- [ ] Locate power outlets in every room (count them)
- [ ] Find the loading dock and trace the path to event spaces
- [ ] Check restroom count and condition (1 per 50 attendees minimum)
- [ ] Assess emergency exits and evacuation routes
- [ ] Take photos of every room, corridor, entrance, and loading area

### After the Visit
- [ ] Score venue on scorecard
- [ ] Note deal-breakers or must-fix items
- [ ] Request floor plan drawings if not already provided
- [ ] Request sample event photos from similar past events
- [ ] Get written confirmation of verbal promises

## RFP Questions for Venues

Send these with your venue inquiry:

1. What is the maximum capacity for each room in theater, classroom, and banquet configurations?
2. What is the daily rental rate? Is there a half-day rate? Multi-day discount?
3. What is included in the rental (AV, WiFi, tables, chairs, linens, cleaning)?
4. What is the WiFi bandwidth? Can we get a dedicated line for streaming?
5. Do you have an in-house AV team or do we need to bring our own?
6. Must we use your in-house caterer or can we bring an external vendor?
7. What is the F&B minimum spend (if applicable)?
8. What are your setup and teardown windows? Can we access the night before?
9. Do you have a loading dock and freight elevator? Dimensions and access hours?
10. What is your cancellation policy? Force majeure clause?
11. What is the parking situation (capacity, cost, valet option)?
12. Is the venue fully wheelchair accessible?
13. What is the latest permitted end time for events?
14. Are there noise restrictions? Curfew for music?
15. Do you have secure overnight storage for equipment?
16. What is your deposit and payment schedule?
17. Can you provide references from similar recent events?
18. What insurance do you require from event organizers?

## Contract Review Checklist

Before signing any venue contract:

- [ ] Rental dates and times match your event plan (including setup and teardown days)
- [ ] Room allocation is specific (named rooms, not "equivalent space")
- [ ] Cancellation terms are acceptable (penalties by date tier)
- [ ] Force majeure clause covers pandemics, natural disasters, government restrictions
- [ ] Liability and insurance requirements are clear and achievable
- [ ] Damage deposit terms and return conditions defined
- [ ] Overtime charges documented (per hour, when they kick in)
- [ ] Exclusivity clauses reviewed (must you use their caterer, AV, security?)
- [ ] Payment schedule is manageable (deposit, milestone payments, final balance)
- [ ] What happens if the venue cancels or double-books (remedies, alternatives)
- [ ] Noise and time restrictions won't conflict with your agenda
- [ ] Guest count flexibility (can you adjust within a range without penalty?)
- [ ] Sub-contracting permissions (can your vendors access the space?)
